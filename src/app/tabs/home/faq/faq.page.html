FAQ If you have any question, kindly select any of the question below. Do I need an account to use Flix Fusion?
No. Currently you don't need to have an account to use Flix Fusion as the database are being publicly shared using The Movie Database (TMDb) API. But you may needed it in future updates.
 Why can I add Movies and TV Shows to Favorites and Watchlist but unable to remove it?
Due to API limitation and older version of TMDb API, we unable to impliment a feature where user could removie Movies or TV Shows to their Favorites and Watchlist. We are currently waiting for an API version 4 update by TMDb.
 Why I can't watch Movies and TV Shows directly on Flix Fusion
Our main purpose for Flix Fusion is to make it free to use and available for all. Streaming on Flix Fusion will wequires a subscription and payment to be done. We are working hard to find colaborators that could offer free straming services.
How does Flix Fusion works?
Flix Fusion works by using an API from The Movie Database (TMDb). All the data are open source and community driven contribution and checked by professionals for authenticity and to avoid data fraud.
